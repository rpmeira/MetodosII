Introduo

O dplyr um pacote de manipulao de dados e sua documentao pode ser encontrada em seu site oficial. Para instal-lo, bastar rodar o comando (nao acrescentar no documento .Rmd, mas rodar no console):

install.packages("dplyr")

E para utiliz-lo, basta rodar o comando (no caso de arquivos .Rmd, acrescentar no incio do documento):

library(dplyr)

O objetivo do pacote ajudar na limpeza dos dados. Ou seja, suas funes ajudam a transformar uma tabela (usualmente um data.frame ou um tibble) crua em uma tabela pronta para a anlise ou para a visualizao.

O pacote possui dezenas de funes, que podem ser agrupadas conforme o elemento da tabela ao qual elas se aplicam (colunas, linhas, grupos de linhas). As mais utilizadas so:

  • Colunas:
    • select(): seleciona as colunas desejadas.
    • mutate(): cria novas colunas ou altera valores de alguma ja existente.
  • Linhas:
    • filter(): seleciona linhas com base em algum critrio de uma varivel (coluna).
    • slice(): seleciona linhas com base em suas posies (ex. as 5 primeiras ou ltimas).
  • Grupos de linhas:
    • summarise(): resume um conjunto de linhas em uma nica (ex. mdia)

Para entender melhor a funcionalidade de cada uma, vamos ver como aplic-las na prtica.

Dados

Todos os exemplos a seguir se baseiam em uma tabela com valores da inflao para cada grupo de produtos. Os dados so do Sistema IBGE de Recuperao Automtica (SIDRA).

Para baixar os dados, vamos utilizar a funo get_sidra() do pacote sidrar. Esse pacote permite baixar tabelas da base de dados do SIDRA. Para maiores informaes, recorrer sua documentao.

Assim como foi feito para o pacote dply, mostrado acima, precisamos instalar e rodar esse pacote:

install.packages("sidrar")

library(sidrar)

Feito isso, para baixar a tabela desejada, utilizamos a funo get_sidra(). O valor do parmetro utilizado (api) obtido no site do SIDRA. A documentao do pacote, apontada acima, explica passo a passo como faz-lo.

library(sidrar)

ipca <- get_sidra(api = '/t/1419/n1/all/v/63,66/p/all/c315/7170,7445,7486,7558,7625,7660,7712,7766,7786/d/v63%202,v66%204')

A funo get_sidra() nos entrega uma tabela no formato data.frame. Para visualizar essa tabela, vamos utilizar a funo paged_table() do pacote rmarkdown.

library(rmarkdown)

paged_table(ipca)

Como possvel observar, a tabela apresenta muitas colunas (variveis) e linhas (instncias), o que torna mais difcil a anlise dos dados.

A partir disso, nas prximas sesses vamos tratar esses dados com a ajuda do pacote dplyr, com o objetivo de obter uma tabela mais enxuta.
