Introduo

Para visualizar a evoluo da inflao de cada grupo de forma separada, preciso transformar esses em variveis (colunas), ao invs de instncias (linhas). Para isso, podemos utilizar a funo pivot_wider(), do pacote tidyr. Adicionalmente, vamos transformar a coluna Ms em uma coluna de datas (ao invs de caracteres).

library(tidyr)

ipca_grupos %>%
  pivot_wider(names_from = `Grupo`, values_from = `Valor Numrico`) %>%
  mutate("Data" = seq(from = as.Date("2012/01/01"), by = "month", length.out = nrow(.))) %>%
  paged_table()

ipca_grupos %>%
  pivot_wider(names_from = `Grupo`, values_from = `Valor Numrico`) %>%
  mutate("Data" = seq(from = as.Date("2012/01/01"), by = "month", length.out = nrow(.))) %>%
  pivot_longer(cols = -1, names_to = "Grupo", values_to = "Valor Numrico") %>%
  ggplot() +
  aes(x = `Data`, y = `Valor Numrico`, color = `Grupo`) +
  geom_line()
